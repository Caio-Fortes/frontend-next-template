# 🔹 🚀 Next/React Template 2026

Estrutura base em React para desenvolvimento de landing pages, com configurações iniciais otimizadas (estilização e organização de pastas).

🔗 [Acesse a aplicação]()

---

## 🚀 Tecnologias utilizadas

**next.js** – construção da interface  
**typescript** – tipagem da linguagem  
**router** – controle de rotas  
**sass** – estilização responsiva e reutilizável  
**vercel** – deploy da aplicação  

---

## 📋 Funcionalidades

- Estrutura base reutilizável para projetos com Next.js  
- Organização de pastas escalável  
- Sistema de rotas integrado  
- Componentização para reutilização de código  
- Estilização modular com Sass  
- Tipagem com TypeScript para maior segurança  
- Layout totalmente responsivo  
- Preparado para SEO  
- Configuração pronta para deploy na Vercel  
- Ambiente com hot reload para desenvolvimento rápido
